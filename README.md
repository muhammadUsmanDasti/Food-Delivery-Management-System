# Food Delivery Management System

A **GUI-based desktop application** designed to manage the complete workflow of a food delivery service. This system provides a centralized interface for administrators to manage users, restaurants, food items, and orders, while allowing customers to place orders seamlessly. Built using **C#**, **.NET**, and **SQL Server**, the project follows a **3-layered architecture (Presentation, Business Logic, Data Access)** for maintainability and scalability.

---

## Features

### Admin Panel
- Add, update, and remove restaurants and food items
- Manage users (customers and staff)
- View all orders with detailed status
- Secure login with role-based access

### Customer Panel
- Browse restaurants and menus
- Place orders and track order status
- Cancel orders if required
- User registration and login

### Technical Highlights
- **3-Layered Architecture**:  
  - **Presentation Layer (UI)**: Windows Forms for interactive GUI  
  - **Business Logic Layer (BLL)**: Handles all core functionalities and validation  
  - **Data Access Layer (DAL)**: Manages database operations with SQL Server
- **Database**: SQL Server integration for storing users, orders, and food items
- **Error Handling & Validation**: Ensures smooth operation and user input validation
- **File Structure**: Modular code for easy maintenance and scalability
